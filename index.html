深度學習底層原理

🧠 類神經網路模擬（Neural Network Playground）

參考自《「生成式⇄AI」：52 個零程式互動體驗》第 3 章與《SimpleLearn - Activation Function 導讀》。 神經網路是深度學習的骨幹。此處模擬一個簡單的 2層 類神經網路正向傳播 (Forward Propagation) 過程。

控制面板

1. 設定輸入變數

輸入 X1 (例如：年齡特徵) 0.50
輸入 X2 (例如：消費紀錄) -0.20

2. 調整權重與偏值

隱藏層神經元 1 偏值 (Bias b1) 0.10
隱藏層神經元 2 偏值 (Bias b2) -0.30

3. 選擇激活函數 (Activation Function)

Sigmoid: σ(x) = 1 / (1 + e^-x)
最經典的激活函數，將輸入壓縮到 (0, 1) 的機率區間，常用於二分類的輸出。
神經網路正向傳播動態流程圖 綠色線: 權重 > 0 | 紅色線: 權重 < 0
輸入矩陣
X = [0.50, -0.20]
B = [0.10, -0.30]
加權和 (Weighted Sum)
z1 = w11*x1 + w21*x2 + b1 = 0.55
z2 = w12*x1 + w22*x2 + b2 = -0.05
神經元輸出 (Activated)
a1 = f(z1) = 0.63
a2 = f(z2) = 0.49
最終預測 Y = 0.54

💡 商情預測理論對應：
  • 特徵提取 (Feature)：x 軸代表特徵（如：廣告預算或點擊量），y 軸代表目標值（如：銷售額）。

💡 詞向量經典運作公式 (Vector Analogy)：
皇上 (King) - 男人 (Man) + 女人 (Woman) = 皇后 (Queen)

Word2Vec 理論： 使用 Skip-gram 模型，在滑動窗口內藉由中心詞預測上下文單詞。 這使得單詞之間隱含的「語意關係」被映射到幾何空間中。 上圖顯示，詞意相近的「蘋果」與「香蕉」、「皇上」與「皇后」，在空間上的歐幾里得距離也最接近。

AI 代理（AI Agent，如 OpenAI 最新發布功能）相較傳統對話具有強大的執行自主權。他們能自主網頁搜索、網頁點擊與讀寫資料，但也伴隨資安風險（如惡意間接提示字注入、隱私數據外洩）。 建議守則： 嚴禁將包含個人財務、未授權公司機密文件提供給對外代理人，並密切防範代理人在執行多步驟操作時，遭到第三方惡意網頁引導執行未授權的敏感操作。
